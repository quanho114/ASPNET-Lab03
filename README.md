# ASPNET-Lab03

Bài thực hành ASP.NET Core 8.0 MVC - Tuần 3 (Lab03) quản lý danh sách sách (Bookstore Catalog).
Các chức năng chính:
- Layout dùng chung
- Partial View (`_BookCard.cshtml`, `_BookSearchResultTable.cshtml`)
- Tag Helpers
- Form tìm kiếm GET
- Form thêm dữ liệu POST
- Model Binding & Validation bằng DataAnnotations
- Quản trị kho (Nhập/bán sách)
- Thống kê kho sách
